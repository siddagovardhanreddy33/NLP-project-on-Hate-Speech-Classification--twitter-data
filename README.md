# NLP Sentiment Analysis using TensorFlow and Keras

This project is a beginner-friendly NLP Sentiment Analysis application built using TensorFlow, Keras, and Streamlit.

The model classifies tweets as Positive or Negative sentiment.

---

# 🚀 Features

- Exploratory Data Analysis (EDA)
- Text Cleaning
- Tokenization
- Padding Sequences
- LSTM Deep Learning Model
- Streamlit Web App
- TensorFlow + Keras
- Beginner Friendly Code
